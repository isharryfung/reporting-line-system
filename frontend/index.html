    Test Case Diagram

    Build a reporting-line diagram to test a case. Click any person to select them as the requester — the resolved reporting line for that person is described in plain wording under the diagram. You can also temporarily change their reporting manager — edits here are never saved to the POC state.

    Pick an action to resolve the selected person's approver line through the routing engine, then add overlays to see how the result changes. Co-head and self-approval-blocked behaviour appear automatically for the chosen action. Overlays here are never saved.
